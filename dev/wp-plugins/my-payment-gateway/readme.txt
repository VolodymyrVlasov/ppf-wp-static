Описание:
Этот плагин добавляет метод оплаты "Мой Метод Оплаты" в ваш магазин на платформе WooCommerce. Вы можете настроить название метода оплаты, короткое описание и ввести текст с реквизитами для отображения на странице оформления заказа.

Основные возможности:
- Добавление кастомного метода оплаты в WooCommerce.
- Настройка названия метода оплаты и короткого описания.
- Возможность ввода текста с реквизитами для метода оплаты.

Установка:
1. Загрузите плагин в папку `wp-content/plugins/` вашего сайта.
2. Активируйте плагин через меню "Плагины" в вашей административной панели WordPress.
3. Перейдите в раздел "Настройки" -> "Оплата" в административной панели WooCommerce, чтобы настроить "Мой Метод Оплаты".

Совместимость:
- Требуется WooCommerce версии 5.0 и выше.
- Протестирован на версии WordPress 5.8.
